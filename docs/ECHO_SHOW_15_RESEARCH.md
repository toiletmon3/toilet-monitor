# Amazon Echo Show 15 — בדיקת התאמה למערכת ToiletMon

**תאריך:** אפריל 2026

## תקציר מנהלים

בדקנו האם מכשיר ה-Amazon Echo Show 15 יכול לשרת אותנו בשני תפקידים:
1. **כמסך קיוסק** — להציג את ממשק הדיווח של ToiletMon בשירותים (במקום טאבלט)
2. **כחיישן ספירת אנשים** — לזהות ולספור כמה אנשים עוברים ליד השירותים

**שורה תחתונה: Echo Show 15 לא מתאים לאף אחד מהשימושים האלה.**
- כמסך קיוסק — יקר יותר ולא אמין ביחס לטאבלט רגיל
- כחיישן ספירה — ה-API של Amazon מוגבל ולא מאפשר ספירת אנשים

להלן הפירוט המלא.

## חלק א — שימוש כמסך קיוסק

### מפרט המכשיר

| תכונה | פרט |
|--------|------|
| מסך | 15.6 אינץ', Full HD (1920×1080) |
| מצלמה | 5 מגה-פיקסל עם זיהוי פנים (Visual ID) |
| מערכת הפעלה | Fire OS (מבוסס אנדרואיד, סגור ע"י Amazon) |
| דפדפן מובנה | Amazon Silk |
| חיבור | Wi-Fi דו-ערוצי |
| מחיר | כ-700–1,000 ש"ח |

### האם אפשר להציג את ToiletMon על המסך?

**כן, אבל עם מגבלות משמעותיות שהופכות את זה ללא מעשי.**

**אופציה א — שימוש בדפדפן Silk המובנה:**
- אפשר לפתוח את הדפדפן עם הפקודה "Alexa, Open Silk" ולגלוש לכתובת הקיוסק
- **הבעיה:** הדפדפן נסגר אוטומטית אחרי כמה דקות וחוזר למסך הבית של Alexa
- קיים מעקף חלקי (סקריפט חיצוני שמונע סגירה), אבל זה פתרון שביר — Amazon יכולה לחסום אותו בכל עדכון

**אופציה ב — התקנת אפליקציית קיוסק (Fully Kiosk Browser):**
- Amazon חסמה התקנת אפליקציות חיצוניות מפברואר 2023
- קיימים מעקפים טכניים, אבל Amazon יכולה לחסום אותם בכל עדכון תוכנה
- לא ניתן לסמוך על זה בסביבת עבודה אמיתית

### השוואה — Echo Show 15 מול טאבלט רגיל

| קריטריון | Echo Show 15 | טאבלט אנדרואיד / iPad |
|-----------|-------------|----------------------|
| הצגת ToiletMon | רק עם מעקפים לא יציבים | עובד מושלם |
| נעילה על מסך אחד | לא קיים | כן — Screen Pinning / Guided Access |
| אפליקציית קיוסק | חסומה ע"י Amazon | נתמכת (Fully Kiosk Browser) |
| יציבות לטווח ארוך | Amazon שולטת ויכולה לשבור | שליטה מלאה שלנו |
| עלות | 700–1,000 ש"ח | 300–500 ש"ח |

**מסקנה:** טאבלט אנדרואיד רגיל עדיף בכל הפרמטרים — זול יותר, אמין יותר, ואנחנו שולטים עליו לגמרי.

## חלק ב — שימוש כחיישן ספירת אנשים

### מה המכשיר יכול לעשות?

ל-Echo Show 15 יש מצלמה עם יכולות זיהוי נוכחות. Amazon חושפת שלושה ממשקי תכנות (API) לנושא:

**1. חיישן נוכחות (Entity Sensing)**
- יכול לזהות אם יש מישהו מול המכשיר (כן/לא)
- נותן מזהה ייחודי לאדם שמזוהה
- מודד את הזווית של האדם ביחס למכשיר
- מדווח כשהאדם הנוכחי עוזב ואדם חדש מגיע

**2. חיישן זיהוי אובייקטים (SmartVision)**
- מזהה שיש "אדם" בשדה הראייה
- שולח התראה לאפליקציית Alexa בטלפון
- לא נותן ספירה — רק "זוהה אדם"

### מה המכשיר לא יכול לעשות?

| מה שאנחנו צריכים | מה שה-Echo Show 15 מספק |
|-------------------|------------------------|
| לספור כמה אנשים עברו | לא — מזהה רק אדם אחד בכל רגע |
| ספירה רציפית 24/7 | לא — עובד רק כשאפליקציה פעילה על המסך |
| לשלוח נתונים למערכת שלנו | לא — אין דרך ישירה לשלוח נתונים לשרת חיצוני |
| לספור מספר אנשים בו-זמנית | לא — מזהה רק "משתמש ראשי" אחד |
| גישה לתמונה מהמצלמה | לא — חסום לחלוטין |

### למה אי אפשר לעקוף את זה?

בתיאוריה אפשר לבנות אפליקציית Alexa שמנסה לספור לפי שינויי זיהוי, אבל:

- **האפליקציה חייבת להיות פעילה על המסך** — ברגע שמישהו אומר "Alexa" או שעובר timeout, הספירה נעצרת
- **מזהה רק אדם אחד בכל רגע** — אם שני אנשים עוברים ביחד, נספור רק אחד
- **הזיהוי לא נועד לספירה** — Amazon בנתה אותו כדי להתאים את התצוגה למשתמש הנוכחי, לא לספור תנועה
- **תלות מלאה ב-Amazon** — הם יכולים לשנות או לבטל את ה-API בכל רגע
- **דורש תשתית נוספת** — חשבון מפתחים ב-Amazon, שרת Lambda ב-AWS, ועלויות חודשיות

**מסקנה:** ה-Echo Show 15 לא מתאים לספירת אנשים.

## חלק ג — חלופות מומלצות לספירת אנשים

אם רוצים לספור כמה אנשים נכנסים ויוצאים מהשירותים, יש פתרונות טובים, זולים ופשוטים הרבה יותר:

### אופציה א — חיישן תנועה אינפרא-אדום (PIR)

| פרט | ערך |
|------|------|
| מה זה | חיישן קטן שמזהה תנועת אדם לפי חום הגוף |
| חומרה | לוח ESP32 + חיישן PIR (דגם HC-SR501) |
| עלות | כ-30–50 ש"ח |
| חיבור | מתחבר ל-Wi-Fi ושולח נתונים ישירות למערכת ToiletMon |
| יכולת | ספירת מעברים — כל פעם שמישהו עובר, המונה עולה |
| אמינות | גבוהה מאוד |
| גודל | קופסה קטנה (כגודל קופסת גפרורים) שמתקינים ליד הדלת |

### אופציה ב — חיישן מרחק לייזר (ToF)

| פרט | ערך |
|------|------|
| מה זה | חיישן שמודד מרחק בלייזר ומזהה מעבר של אדם |
| חומרה | לוח ESP32 + חיישן VL53L1X |
| עלות | כ-50–80 ש"ח |
| חיבור | Wi-Fi ישירות למערכת ToiletMon |
| יכולת | ספירה מדויקת + הבחנה בין כניסה ליציאה |
| אמינות | גבוהה מאוד, מדויק יותר מחיישן PIR |
| גודל | קופסה קטנה שמתקינים במשקוף הדלת |

### אופציה ג — מחסום אינפרא-אדום (beam break)

| פרט | ערך |
|------|------|
| מה זה | שני חיישנים בצדי הדלת — כשמישהו עובר ביניהם, הקרן נשברת |
| חומרה | לוח ESP32 + זוג חיישנים אינפרא-אדום |
| עלות | כ-40–60 ש"ח |
| חיבור | Wi-Fi ישירות למערכת ToiletMon |
| יכולת | ספירה כיוונית — מבחין בין כניסה ליציאה (שני חיישנים = כיוון) |
| אמינות | גבוהה, פשוט להתקנה |

### טבלת השוואה

| קריטריון | Echo Show 15 | חיישן PIR | חיישן לייזר (ToF) |
|-----------|-------------|-----------|-------------------|
| ספירת אנשים | לא | כן | כן |
| ספירה רציפית 24/7 | לא | כן | כן |
| עלות | 700–1,000 ש"ח | 30–50 ש"ח | 50–80 ש"ח |
| שליחת נתונים למערכת | לא | כן, ישירות | כן, ישירות |
| הבחנה כניסה/יציאה | לא | דורש שני חיישנים | מובנה |
| מורכבות התקנה | גבוהה | נמוכה | נמוכה-בינונית |
| תלות בספק חיצוני | כן (Amazon) | לא | לא |
| תאימות ל-ToiletMon | לא | כן — תשתית קיימת | כן — תשתית קיימת |

### מה צריך לפתח?

המערכת שלנו (ToiletMon) כבר מכילה תשתית בסיסית לחיישנים בקוד. צריך להשלים שלושה דברים:
1. **נקודת קליטה בשרת** — כתובת API שהחיישן שולח אליה את נתוני הספירה
2. **קוד לחיישן** — תוכנה קטנה שרצה על לוח ה-ESP32 ומדווחת מעברים דרך ה-Wi-Fi
3. **תצוגה בדשבורד** — גרף בלוח הבקרה של המנהל שמראה כמה אנשים נכנסו לשירותים לפי שעות/ימים

## סיכום והמלצות

### למסך קיוסק — להישאר עם טאבלט רגיל
- זול יותר (300–500 ש"ח לעומת 700–1,000 ש"ח)
- עובד מושלם עם אפליקציית קיוסק (Fully Kiosk Browser)
- נעילת מסך מובנית (Screen Pinning באנדרואיד / Guided Access באייפד)
- ToiletMon כבר תומך בטאבלטים לגמרי

### לספירת אנשים — חיישן ESP32 (PIR או ToF)
- עלות נמוכה מאוד (30–80 ש"ח ליחידה)
- ספירה רציפית 24 שעות ביממה, 7 ימים בשבוע
- שליחת נתונים ישירות למערכת ToiletMon דרך ה-Wi-Fi של המבנה
- התקנה פשוטה — קופסה קטנה ליד דלת השירותים
- תשתית בסיסית כבר קיימת בקוד המערכת

### Echo Show 15 — לא מומלץ
- לא מתאים כקיוסק — אין מצב נעילה, הדפדפן נסגר לבד, יקר
- לא מתאים לספירת אנשים — מזהה רק אדם אחד, עובד רק כשאפליקציה פעילה, אי אפשר לשלוח נתונים החוצה
- תלות מלאה ב-Amazon — כל עדכון תוכנה שלהם יכול לשבור הכל
