מדריך מהיר — חיבור ראדאר HLK-LD2450 ל-ESP32 DevKitV1

ראדאר נוכחות mmWave 24GHz (מעקב עד 3 מטרות) · תקשורת UART · לוח NodeMCU-ESP32 בן 30 פינים

1 החיישן — HLK-LD2450

  • VCC מתח כניסה 5V (מקבל גם 3.3V)
  • GND הארקה (0V)
  • TX יציאת נתונים טורית (UART)
  • RX כניסת נתונים טורית (UART)

ראדאר 24GHz — מזהה מיקום ותנועה ומחזיר קואורדינטות X/Y של עד 3 מטרות ב-UART 256000 baud. זהו חיישן טורי — לא יציאת HIGH/LOW כמו PIR.

2 חיווט (טבלה)

LD2450הלוח שלך
VCC→3V3
GND→GND
TX→RX2 (GPIO16)
RX→TX2 (GPIO17)

⚠️ TX/RX מוצלבים! TX של החיישן → RX2 של הלוח, ולהפך.

3 הזנת מתח — שים לב!

בלוח ה-30 פינים הזה אין פין שכתוב עליו "5V". מקורות המתח היחידים הם:
• VIN = ה-5V של הלוח (פינה שמאלית-עליונה)
• 3V3 = 3.3V (פינה שמאלית-תחתונה)

אם ה-VIN חסום/תפוס → הזן את הראדאר מ-3V3 (ה-LD2450 מקבל 3.3V). אם לא יציב — 5V חיצוני עם GND משותף.

4 קוד לבדיקה (Arduino / Serial2)

// LD2450 TX->GPIO16(RX2), RX->GPIO17(TX2)
#define RXD2 16
#define TXD2 17
uint8_t buf[30];

int16_t coord(uint8_t lo, uint8_t hi){
  int16_t v = ((hi & 0x7F) << 8) | lo;
  return (hi & 0x80) ? v : -v;  // mm
}
void setup(){
  Serial.begin(115200);
  Serial2.begin(256000, SERIAL_8N1, RXD2, TXD2);
}
void loop(){
  if (Serial2.available() >= 30){
    Serial2.readBytes(buf, 30);
    if (buf[0]==0xAA && buf[1]==0xFF){
      int16_t x = coord(buf[4], buf[5]);
      int16_t y = coord(buf[6], buf[7]);
      Serial.printf("X=%d mm  Y=%d mm\n", x, y);
    }
  }
}

💡 טיפים חשובים

  • TX/RX תמיד מוצלבים: TX→RX2, RX→TX2.
  • מהירות ה-UART חייבת להיות 256000 baud, 8N1.
  • הלוגיקה של החיישן 3.3V — בטוח לחיבור ישיר ל-ESP32.
  • אין פין 5V נפרד — ה-5V כאן נקרא VIN.
  • אם VIN חסום — הזן מ-3V3 (הראדאר מקבל 3.3V).
  • לפירוק פריים אמין השתמש בספריית LD2450 מוכנה.
