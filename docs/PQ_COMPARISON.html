📊 סטטיסטיקות וניתוח נתונים

קטגוריה / מדד דרישת PQ תעשייתית ToiletMon — מצב נוכחי סטטוס
זמן תגובה ממוצע (Resolution Time) מדידת זמן מרגע דיווח/חיישן עד סגירה. יעד <15 דקות נמדד: reportedAt → resolvedAt. ממוצע 30 יום + median (p50), p90. יעד SLA מותאם: 10/15/20/30 דקות ✅ מלא
אחוז עמידה ב-SLA % תקלות שנפתרו בתוך זמן היעד. יעד >85% slaPercent מחושב בזמן אמת עם target מותאם. תצוגה: gauge + within/total ✅ מלא
חלוקת תקלות לפי סוג (Issue Frequency) פירוט כמות דיווחים לפי קטגוריה (ניקיון, ריח, חוסר אספקה וכו') issue-frequency API: count + avgResolution per type. תרשים עמודות באנליטיקס ✅ מלא
דפוסי שעות שיא (Hourly Patterns) ניתוח תנועה ושימוש לפי שעה ביום 24 buckets עם count per hour + תרשים. day-of-week analysis (7 ימים) ✅ מלא
ביצועי עובדים (Cleaner Performance) כמות פתרונות, זמן ממוצע, השוואה בין עובדים leaderboard: totalResolved + avgResolutionMinutes per cleaner + idle detection בדו"ח יומי ✅ מלא
זיהוי נקודות חמות (Hotspots) אילו שירותים דורשים הכי הרבה טיפול patterns API: top 5 restrooms by incident count + heatmap API (server ready, UI planned) ✅ מלא
דו"ח יומי אוטומטי סיכום יומי לניהול עם KPIs Email daily report: incidents/resolved/SLA/top issues/hotspots/cleaners/idle list ✅ מלא
ייצוא דוחות (PDF/CSV) ייצוא נתונים לגורמים חיצוניים / הנהלה ב-Roadmap (v1.2.0). כרגע אין ייצוא ידני ❌ חסר

🔧 ניהול תפעולי ותגובה

קטגוריה דרישת PQ תעשייתית ToiletMon — מצב נוכחי סטטוס
התראות זמן אמת (Real-time Alerts) Push/SMS/email מיידי על תקלה חדשה WebSocket push מיידי + toast בממשק מנהל. Web Push (PWA) לטלפון בפיתוח (v1.1.0) ⚠️ חלקי
אסקלציה אוטומטית העלאת חומרה אוטומטית לפי זמן שעבר ללא טיפול רמות אסקלציה בקפיצות 5 דקות (configurable). Urgent alerts בדשבורד ✅ מלא
הקצאת משימות לעובדים שיוך תקלה לעובד ספציפי + מעקב assignedCleaner על incident + מעקב status. Bulk-resolve siblings ✅ מלא
מעקב נוכחות עובדים Check-in/out + שעות עבודה + mismatch detection Check-in/out מקיוסק פיזי. Mismatch alerts (30 דקות+ ללא פעולה). פאנל "כרגע בעבודה" ✅ מלא
תזמון ניקיון דינמי (Occupancy-based) ניקיון על-פי עומס בפועל (לא schedule קבוע) תגובתי לדיווחים. אין תזמון יזום על בסיס occupancy/footfall ⚠️ חלקי

📡 חיישנים ו-IoT

קטגוריה דרישת PQ תעשייתית ToiletMon — מצב נוכחי סטטוס
חיישן תפוסה (Occupancy Sensor) ספירת כניסות/יציאות, מפת תאים פנויים בזמן אמת לא קיים. המערכת מבוססת דיווח אנושי (קיוסק) — לא חיישנים פיזיים ❌ חסר
חיישן ריח / איכות אוויר (NH₃) התראה אוטומטית ברמת אמוניה >25ppm לא קיים. ב-Roadmap (feature/hardware-sensor-device branch) ❌ חסר
חיישן רמת מתכלים (סבון/נייר) ניטור רמת מילוי + התראה אוטומטית בירידה מתחת ל-20% לא קיים. ניתן לדווח ידנית דרך הקיוסק ❌ חסר
חיישן רטיבות רצפה / דליפה זיהוי מים על הרצפה + התראה מיידית לא קיים. ניתן לדווח ידנית ("רצפה רטובה") ❌ חסר
Heartbeat / Device Monitoring מעקב online/offline על כל מכשיר ✅ מלא: isOnline, lastHeartbeat, offline device list + alerts ✅ מלא
